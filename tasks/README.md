# tasks/ — local working files

This directory holds **per-session working files** used by the Workflow Standard. They are **local
working state** and are intentionally **git-ignored** (kept off the public repository for privacy).

- `todo.md` — the current session's execution plan (archived at session end).
- `lessons.md` — running lessons-learned log.

Recreate them locally as needed; they are not part of the published project.
